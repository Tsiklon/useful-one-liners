Useful AWK stuff

Using AWK to search output ala grep;
(These also work when stdin is piped to awk)

# awk '/string to match/' /path/to/file  - Searching for a single string 
# awk '/string 1|string 2/' /path/to/file     - searching for more than one string

# awk '!/string to exclude/' /path/to/file    - Excluding a single string
# awk '!/string 1|string 2/' /path/to/file    - excluding more than one string

Searching for a single string and outputing only the desired column 
# awk '/string to match/{print $1}' /path/to/file    - match all lines matching the string and output field 1 (default field separator is ' ' (a single space))
# awk -F '-' '/string to match/{print $3}' /path/to/file   - match all lines matching the string and output field 3, with field separator '-'
